## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

```

We are using Pascal case for components folder and files and kabab case for all other folders and files.

e.g

components
  -> Title
    -> index.ts
pages
  -> LoginPage
     -> index.tsx
     -> TopComponent.tsx
     -> BottomComponent.tsx
